# Master of Detailing — Ceramic Coating Asset Pack

This pack contains:

- the three product images supplied directly by Luke
- optimized WebP derivatives for those supplied images
- the full ceramic-coating product manifest
- official Angelwax product-page references for the remaining coating options
- neutral local placeholders for development previews
- a ready-to-copy `product-content-snippet.js`

## Important image-use note

The official Angelwax product-page image sources are listed for reference and approval workflows.

Do not publish third-party product images unless Matthias has permission to use them, receives an approved supplier media pack, or supplies authorised local copies.

The local placeholders marked `AUTHORIZED PRODUCT IMAGE PENDING` are intended for development preview only.

## Installation

Extract this ZIP into:

`C:\Users\User\master-of-detailing`

Allow Windows to merge the included `public` folder with the existing project `public` folder.

Then use:

`product-content-snippet.js`

as the source of truth when adding the Ceramic Coating Options section.
