# Master of Detailing — Website Asset Pack

This ZIP has been prepared from the uploaded WeTransfer folder for the Master of Detailing website.

## What is included

- `public/photos/gallery/` — 8 selected real workshop images, renamed for the live website gallery
- `public/photos/before-after/` — 4 verified before-and-after pairs for the Results section
- `public/photos/optional-results/` — additional comparison images that can be enabled later
- `public/photos/hero/README.txt` — reminder to keep Luke's existing `hero-main.png`
- `content-data-snippet.js` — ready-to-copy data entries and feature flags
- `preview/asset-contact-sheet.jpg` — a quick preview of the prepared asset pack

## Installation

Extract this ZIP into:

`C:\Users\User\master-of-detailing`

Allow Windows to merge the included `public` folder with the existing project `public` folder.

Keep your custom hero file at:

`C:\Users\User\master-of-detailing\public\photos\hero\hero-main.png`

Then update `src/data/content.js` using the included `content-data-snippet.js`.
