# Master of Detailing — Auto Graph Detailing Product Pack

This pack contains the three supplied Auto Graph Detailing stock photos and website-ready files.

## Included

- `public/products/auto-graph/`
  - three original PNG images
  - three optimized WebP derivatives
- `auto-graph-product-manifest.json`
- `auto-graph-product-manifest.csv`
- `auto-graph-content-snippet.js`
- `social-post-caption.txt`

## Suggested website use

Add a premium **Products Now Available** section lower on the homepage or as a compact supporting block near the Contact section.

Recommended structure:

- Wheel Care
- Interior Care
- Exterior Care
- WhatsApp CTA for prices and product recommendations

Keep the section advisory and WhatsApp-first. Do not make it look like a full online shop unless Matthias later wants ecommerce.

## Installation

Extract this ZIP into:

`C:\Users\User\master-of-detailing`

Allow Windows to merge the included `public` folder with the existing project `public` folder.

Then use:

`auto-graph-content-snippet.js`

as the content source when adding the website section.
